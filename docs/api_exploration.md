## Mountain weather availability

Mountain weather is NOT exposed via `/v2/public/products?avalanche_center_id=NWAC`.

Phase 2 fallback options to investigate before implementation:

- The NWAC mountain weather page (https://www.nwac.us/mountain-weather-forecast/) likely renders prose from a separate backend route. Inspect network calls.
- The avalanche.org product schema may expose mountain weather under a different filter. Try variants like `/v2/public/products?avalanche_center_id=NWAC&product_type=mountain_weather`.
- If no public endpoint exists, scrape the rendered HTML and treat the result as a verbatim narrative product.
